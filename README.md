# FIFA Player and Team Analysis

## Overview

This project explores and analyzes FIFA player and team data using Python and data analysis libraries. The goal is to uncover insights about player attributes, wages, market values, team strengths, and overall trends within the FIFA dataset.

## Objectives

* Perform data cleaning and preprocessing on FIFA datasets.
* Analyze player characteristics such as age, nationality, overall rating, and potential.
* Explore player wages and market values.
* Compare teams based on player ratings and squad composition.
* Visualize trends and relationships using charts and graphs.
* Derive meaningful insights that can support player scouting and team evaluation.

## Technologies Used
